# ApollonAssignment

* Coroutines 
* Room
* MMVM
* Clean Architecture
* Retrofit

Due to time constarints and some unavoidable circumstances I was not able to include

* Android compose
* Dagger
* JetPack navigation library
* TDD
* BDD
